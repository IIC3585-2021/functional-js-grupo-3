# functional-js-grupo-3

## Ejecución de código

    node index.js

## Formato inputs
 
 * Los nombres de los jugadores debe ser separados por coma. 
 * El juego solo comenzará cuando se ingresen más de dos jugadores.

    nombre_jugador1, nombre_jugador2, ...

* Las jugadas deben ser ingresadas separadas por coma. 
* Se deben ingresar 3 jugadas.
* En caso de ser puntajes, se ingresan de la forma [multiplicador, puntaje].
* En caso de ser alguno de los caso especiales, se ingresa con comillas(pueden ser dobles o simples), es decir, el single bull se ingresa como "SB", double bull como "DB" y el nulo como "Null".
